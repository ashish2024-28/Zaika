# 🍽️ Zaika – Restaurant Menu Web App

Zaika is a simple restaurant menu web application that displays food items dynamically using HTML, CSS, JavaScript, and XML.  
This project demonstrates how menu data stored in an XML file can be fetched and displayed on a webpage.

---

## 📌 Features
- Display restaurant menu items
- Veg / Non-Veg food categories
- Menu data stored in XML
- Dynamic content loading using JavaScript
- Clean and simple user interface

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- XML

---

## 📂 Project Structure
```
Zaika
│
├── index.html      # Main webpage
├── style.css       # Styling file
├── app.js          # JavaScript logic
├── data.js         # Data handling
└── menu.xml        # Menu data
```

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in any web browser

---

## 📖 Project Description

The Zaika Restaurant Web App loads food menu data from an XML file and displays it on the webpage using JavaScript.  
It helps demonstrate how structured data can be integrated with frontend technologies.
